### 简介
nodejs+express+mysql操作数据库，简单实用，包括增删改查、图片上传、分页、模糊查询、排序、接口路由等

### 技术栈
nodejs，express，mysql

## 接口示例
```
//查询某个用户信息
router.get("/person",(req,res) => {
	db.query('select * from user where user_id = ?', req.query.user_id, (err, result)=>{
		if(err){
			throw new Error(err)
		}else{
			if(!result[0]){
				res.json({
					code:250,
					msg:"用户不存在"
				})
			}else{
				res.json({
					code:200,
					msg:'ok',
					data:result[0]
				})
			}
		}
	})
})
```
